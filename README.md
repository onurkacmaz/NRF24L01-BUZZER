Using NRF24L01 to play sound from the buzzer with wireless communication.

RF24 library
LiquidCrystal library
SPI library

# Required objects 

16x2 LCD Screen
Buzzer

NRF24L01 Modul

Resistance
Grove Infrared Temperature Sensor
Breadbord
